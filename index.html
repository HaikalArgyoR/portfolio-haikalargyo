My Projects

FiMa23 (3D Filament Maker)

FiMa23 is an innovative filament making machine for 3D printers, designed to be easy to operate and highly flexible. Its control core is built around a dual-core ESP32 module running an RTOS (Real-Time Operating System), allowing all processes—from extruder heating and temperature measurement to filament extrusion—to run in parallel without interruption. A precision-scale feedback system uses a digital temperature sensor at the extrusion head to ensure a consistently stable melt temperature, delivering uniform filament quality.

Arduino IDE RTOS

EcoBoost (Organic waste processing device)

A prototype Arduino based organic waste processing unit designed to automate the efficient crushing and fermentation of organic refuse. With a focus on temperature stability and material quality monitoring, EcoBoost ensures the decomposition process runs optimally to produce high-quality compost.

Zero-Waste Arduino

Trainer Board PLC and Pneumatik

A comprehensive experimental apparatus specifically designed for industrial automation education at the vocational high-school level. Featuring modular, easy-to-assemble practice units, this device combines PLC-based control logic with pneumatic actuator circuits, allowing students to learn control concepts alongside real-world industrial applications.

PLC Pneumatic

Tower Copter

Tower Copter is a control systems learning prototype designed to regulate height and maintain motion stability in a controlled manner. The system is equipped with a Unity-based user interface to simplify monitoring and control. The control logic is implemented using a PID (Proportional-Integral-Derivative) controller, which continuously adjusts the motor speed based on feedback from an ultrasonic sensor to maintain the desired height and ensure stable flight.
